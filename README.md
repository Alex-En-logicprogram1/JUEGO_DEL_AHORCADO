# Juego del Ahorcado

## Integrantes

Alex Encalada

## Objetivo

Desarrollar un juego del Ahorcado que permita al usuario adivinar una palabra mediante el ingreso de letras, aplicando estructuras lógicas de programación y buenas prácticas de desarrollo.

## Funcionalidades

- Menú principal
- Selección aleatoria de palabras
- Validación de letras
- Conteo de intentos
- Mensajes de victoria y derrota
- Reinicio del juego
